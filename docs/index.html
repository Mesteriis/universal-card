💻 Code Examples

Start with copyable YAML in under a minute

lovelace.yaml 
type: custom:universal-card
title: Lighting
subtitle: Living Room
icon: mdi:lightbulb-group
entity: light.living_room
body:
  cards:
    - type: light
      entity: light.living_room
    - type: light
      entity: light.bedroom
Grid Layout 
type: custom:universal-card
title: Dashboard
grid:
  columns: 3
  gap: 16px
body:
  cards:
    - type: sensor
      entity: sensor.temperature
      colspan: 2
    - type: sensor
      entity: sensor.humidity
    - type: gauge
      entity: sensor.power
      rowspan: 2
Tabs Mode 
type: custom:universal-card
title: Smart Home
body_mode: tabs
tabs:
  - label: Lights
    icon: mdi:lightbulb
    cards:
      - type: light
        entity: light.room
  - label: Climate
    icon: mdi:thermometer
    cards:
      - type: thermostat
        entity: climate.ac
Modal Mode 
type: custom:universal-card
title: Settings
icon: mdi:cog
body_mode: modal
modal:
  width: 500px
  backdrop_blur: true
body:
  cards:
    - type: entities
      entities:
        - input_boolean.setting_1
        - input_number.setting_2
Theme 
type: custom:universal-card
title: Neon Card
theme: neon
expanded: true
body:
  cards:
    - type: entities
      entities:
        - light.room
        - switch.fan
        - sensor.temperature

📦 Installation

Choose the installation path that fits your setup

🎨 HACS

  1. Open HACS → Frontend
  2. Select "+" and search for "Universal Card"
  3. Click "Download"
  4. Restart Home Assistant

📁 Manual Install

  1. Download the latest package from GitHub Releases
  2. Copy universal-card.js and the lazy/ directory into /config/www/universal-card/
  3. Add the Lovelace resource:
    /local/universal-card/universal-card.js (module)
  4. Refresh the page and clear browser cache if needed
